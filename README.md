# MovieStudio
TMDB Api MovieApp
TMDB Api : https://developers.themoviedb.org/3/getting-started/introduction <br />
See Top Rated and Recent Popular Movies.

## Content
1) Retofit
2) GSON
3) Coil
4) Zoom RecyclerView
5) OkHttp
6) Lifecycle & Kotlin Components
7) Navigation
8) ViewModel / LiveData
9) Coroutines
10) Repository
11) Room Database
